# SpringBoot-MusicWeb
Base on Springboot and Vue to create a musicweb

# 网站访问
    http://momusic.top/?#/mainpage

# 1.1项目环境
### 请自行安装对应版本 ###
前端:

    -Node v18.15.0
    
    -Vue Cli 5.0.8
    
    -Vue3
    
后端:

    maven 3.6.1
    
    java 18.0.2.1
    
    springboot 3.0.3
# 1.2项目启动

    -npm run serve
